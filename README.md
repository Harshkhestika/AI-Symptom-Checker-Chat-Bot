# 🩺 AI Symptom Checker Chatbot

An intelligent web-based AI Symptom Checker Chatbot built using **HTML, Tailwind CSS, JavaScript, and Gemini AI API**. The chatbot helps users interact through text and voice to discuss symptoms, health concerns, and receive AI-generated responses.

---

## 🚀 Features

* 💬 AI-powered symptom checking and health guidance
* 🎙️ Voice input using Speech Recognition API
* 🌙 Dark Mode / Light Mode support
* 📱 Fully responsive design
* 📝 Chat history stored in Local Storage
* ⚡ Real-time AI responses
* 📚 Previous conversation management
* 🎨 Modern Tailwind CSS user interface
* 🔄 New Chat and History Management
* ⏰ Live clock display

---

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* JavaScript (ES6)
* Gemini AI API
* Local Storage API
* Web Speech API

## 🔑 API Configuration

Replace the API details inside the script section:

```javascript
const API_URL = "YOUR_GEMINI_API_URL";
const API_KEY = "YOUR_GEMINI_API_KEY";
```

Example:

```javascript
const API_URL =
"https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent";

const API_KEY = "YOUR_API_KEY";
```

## ▶️ How to Use

1. Open the application.
2. Type your symptoms or health-related questions.
3. Click **Send**.
4. Alternatively, use the microphone button for voice input.
5. Receive AI-generated responses instantly.
6. View previous conversations in Chat History.
